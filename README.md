# KVault

A simple self-hosted password manager built with React, Tailwind, Express, and MongoDB.

## How to run

- Start MongoDB locally.
- In `/backend`, run:  
  `npm install`  
  `node server.js`
- In the main folder, run:  
  `npm install`  
  `npm run dev`
